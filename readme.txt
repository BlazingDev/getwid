== Description ==

A collection of beautiful and customizable blocks.

* Section
* Icon
* Icon Box
* Banner
* Image Slider
* Media and Text Slider
* Google Maps
* Accordion
* Toggle
* Tabs
* Image Stack Gallery


== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
